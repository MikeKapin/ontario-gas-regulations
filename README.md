# Ontario Gas Regulations and Code Adoption Documents

Ontario Regulations and Code Adoption Documents (CAD) for propane and gaseous fuel safety, handling, and installation.

## Overview

This repository contains the complete set of Ontario regulations governing gas fuel safety and handling, including Code Adoption Documents that adapt CSA standards for Ontario implementation.

## Regulations

### Ontario Regulation 211/01 - Propane Storage and Handling

Governs the storage, handling, and transportation of propane in Ontario.

**Key Topics:**
- Storage facility requirements
- Container specifications
- Safety equipment and procedures
- Inspection and maintenance
- Emergency response procedures

### Ontario Regulation 212/01 - Gaseous Fuels

Governs the installation, maintenance, and operation of natural gas and propane equipment in Ontario.

**Key Topics:**
- Appliance installation requirements
- Venting and combustion air
- Piping and connectors
- Safety devices and controls
- Maintenance and inspection standards

### Ontario Regulation 215/01 - Fuel Certificates

Governs the qualification and certification of fuel handlers and technicians in Ontario.

**Key Topics:**
- Certificate types and categories
- Training and examination requirements
- Competency standards
- License maintenance and renewal
- Professional responsibilities

## Code Adoption Documents (CAD)

### Propane CAD

Amendments and clarifications to CSA B149.2-25 for Ontario implementation of propane standards.

### Gaseous Fuels CAD

Amendments and clarifications to CSA B149.1-25 for Ontario implementation of natural gas and propane standards.

## Compliance Framework

All gas safety and installation content must comply with:

1. **Ontario Regulation 211/01** - Propane Storage and Handling
2. **Ontario Regulation 212/01** - Gaseous Fuels
3. **Ontario Regulation 215/01** - Fuel Certificates
4. **CSA B149.1-25** - Natural Gas Installation Code
5. **CSA B149.2-25** - Propane Storage and Handling Code
6. **Transport Canada Regulations** - Dangerous Goods Transportation
